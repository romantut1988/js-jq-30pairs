1. Сверстать лендинг. Внешне сверстанный сайт должен быть полностью идентичен макету. Можно не добиваться pixel perfect, но правильная верстка по макету очень важна. Адаптивную версию можно сделать только для хедера и первого блока лендинга (по желанию - всего лендинга) Фреймворки подключать не нужно, функционал реализуется на чистом CSS и JS. Из готовых инструментов допускается использование: jQuery wow js плагина для лайтбокса 2. Добавить эффекты при наведении на кнопки, анимации блокам с wow js на свое усмотрение. 3. Реализовать стандартную валидацию формы внизу страницы (и + еще в поле имя нельзя вводить точки, они просто не вводятся) 4. При отправке формы отправляется на сервер ajax-запрос с введенными данными. На странице выводим текст “Спасибо за заказ” в попапе. (Использовать фейковые запросы без реального бэкенда) 5. При нажатии на картинки товаров они открываются в режим просмотра на весь экран (даже если картинки маленькие в макете). Можно реализовать с помощью плагина.
